# Guia rápido

A documentação principal do projeto está em [`README.md`](README.md).

Use este arquivo apenas como atalho rápido:

```bash
npm install
npm run dev
npm run test
npm run test:watch
npm run build
npm run lint
npm run format:check
```

API local:

```text
http://localhost:3000/api/tasks
```

Se o Next iniciar em outra porta, use a porta mostrada no terminal.
